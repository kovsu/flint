This rule checks that the `homepage` property is a string containing a valid URL.

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
	"homepage": "example@flint.fyi"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
	"homepage": "https://flint.fyi"
}
```

</TabItem>
</Tabs>

## When Not To Use It

If you don't need compliance with npm's `homepage` field specification, you can disable this rule.
For example, if you use a different package manager that treats the field differently, this rule may not be correct for you.
