This rule checks that the `private` property is a boolean.

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
	"private": "true"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
	"private": true
}
```

</TabItem>
</Tabs>

## When Not To Use It

If you don't need compliance with npm's `private` field specification, you can disable this rule.
For example, if you use a different package manager that treats the field differently, this rule may not be correct for you.
